A personal portfolio website for **John Enrico D. Lauron**, a Computer Engineering student focused on practical software and hardware-aware project work.

The portfolio highlights projects across full-stack web development, embedded systems, database-backed applications, automation workflows, and computer engineering coursework. It is built as a lightweight static site with a dark, minimal, terminal-inspired interface.

## Features

- Responsive layout for desktop and mobile screens
- Minimal terminal-inspired UI with green accent styling
- Project showcase section with clickable repository cards
- Technical experience page for academic and project background
- Social links integration for GitHub, LinkedIn, email, and Facebook
- GitHub Pages deployment
- Accessibility-focused navigation with semantic HTML and visible focus states

## Technology Stack

## Project Highlights

- **Carolinian Events Tracker**: University event discovery and organizer workflow project.
- **Aquatic Hatchery Monitoring System**: Hardware-aware monitoring concept for aquatic hatchery conditions.
- **Flask Gemini Chat App**: Flask-based AI chat application using Gemini API integration.
- **Machine Efficiency Dashboard**: Google Sheets and Apps Script dashboard for machine performance reporting.
- **KwarTrack**: Java-based financial tracking application for personal spending records.
- **T3via**: C trivia quiz application focused on console interaction and scoring logic.
- **8086 Fish Hatchery Control System**: Microprocessor-based control concept using x86 Assembly and 8086 fundamentals.

## Local Development

Clone the repository:

```bash
git clone https://github.com/ScarXiFy/ScarXiFy.github.io.git
cd ScarXiFy.github.io
```

Open the site directly:

```bash
open index.html
```

On Windows, you can double-click `index.html`, or open the folder in VS Code and use the Live Server extension for a local preview.
